🔥 Firebase Setup
APEX uses your own Firebase project so your data stays private. This is a one-time setup. Your config is saved locally in the browser.
Step 01 / Create Firebase Project
Go to Firebase Console
Visit console.firebase.google.com → Create project → Disable Google Analytics (optional) → Continue.
Step 02 / Enable Google Auth
Authentication → Sign-in method
In your project: Authentication → Get Started → Google → Enable → Save. Add your domain to Authorized Domains.
Step 03 / Create Firestore Database
Firestore Database → Create
Choose Start in production mode. Select region close to you. Then go to Rules tab and paste:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth != null && request.auth.uid == userId; } } }
Step 04 / Get Your Config
Project Settings → Web App → Register App
Click the web icon (</>), name your app, then copy the firebaseConfig object values below:
🔒 Privacy Note
Your Firebase config is stored in localStorage on this browser only. It is never sent to any APEX server. You own your data.
